# Keycomerce Bot — Financeiro no Telegram

Bot de gestão financeira da Keyco via Telegram.

## Variáveis de ambiente necessárias no Railway:
- `TELEGRAM_TOKEN` — token do seu bot (@BotFather)
- `ANTHROPIC_KEY` — sua chave da API Anthropic

## Funcionalidades:
- Contas a pagar — lançar, consultar, marcar pago
- Contas a receber — lançar, consultar, marcar recebido
- Orçamentos — criar, aprovar, recusar
- Leitura de boleto por foto/PDF com IA
